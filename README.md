# AnimalPic
​
## サイト概要
### サイトテーマ

動物限定写真投稿コミュニティサイト
​
### テーマを選んだ理由

 もともと動物が好きで、SNSでも動物についての写真動画を検索することが多いため、動物だけが見れるサイトがあったらいいなと思ったからです
​また、近頃のペットブームでペットや動物園水族館など動物への関心、需要についても高まっていると思いました
### ターゲットユーザ

- 動物を飼っている人
- 動物が好きな人

​
### 主な利用シーン
- 自分のペットの写真を共有したいとき

- 動物の写った写真を共有したいとき
